Step 1 — Go to console.firebase.google.com and create a new project (free Spark plan)
Step 2 — In your project: click Build → Realtime Database → Create Database → choose "Start in test mode"
Step 3 — Click the gear icon (Project Settings) → scroll to "Your apps" → click the Web icon → register app
Step 4 — Copy the firebaseConfig object → paste it at the top of this file replacing the placeholder values
Step 5 — Push the updated file to GitHub Pages → your progress now syncs across every device in real time
